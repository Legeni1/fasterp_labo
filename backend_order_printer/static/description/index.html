Overview

This Module help you to print quickly and easily QWeb with HTML type.

Configuration

  • Clone addon from git (extract if needed)
  • Upload addon to odoo addons folder
  • Restart odoo service
  • Go to Odoo => Apps => Update Apps List if needed
  • Search "Print Orders Directly From Backend"
  • Install app
